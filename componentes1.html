Listado por Categorías

1. Componentes Pasivos

Estos componentes no generan energía, solo la almacenan, disipan o modifican señales eléctricas.

  • Resistencia
  • Condensador
  • Bobina
  • Fusible
  • Potenciómetro
  • LDR (resistencia dependiente de luz)
  • Termistor
  • Varistor
  • Cristal de cuarzo
  • Zener

2. Componentes Activos

Son capaces de amplificar señales eléctricas y controlar el flujo de corriente.

  • Diodo
  • LED
  • Transistor
  • Tiristor
  • Triac
  • Regulador de voltaje
  • Oscilador

3. Circuitos Integrados

Son dispositivos que integran múltiples componentes electrónicos en un solo chip.

  • Circuito Integrado
  • Microcontrolador
  • Microprocesador

4. Conmutación y Protección

Controlan el paso de la corriente eléctrica o protegen al circuito ante fallas.

  • Interruptor
  • Relé
  • Optoacoplador

5. Actuadores

Transforman energía eléctrica en movimiento o sonido.

  • Motor DC
  • Motor paso a paso
  • Servomotor
  • Zumbador piezoeléctrico
  • Buzzer

6. Sensores

Detectan variaciones físicas del entorno (como temperatura o luz) y las transforman en señales eléctricas.

  • Sensor de temperatura
  • Sensor de humedad
  • Sensor de gas
  • Sensor PIR (infrarrojo pasivo)
  • Sensor magnético
  • Sensor ultrasónico
  • Acelerómetro
  • Giroscopio

7. Pantallas y Visualización

Permiten mostrar información visual como texto, números o gráficos.

  • Display de 7 segmentos
  • Pantalla LCD
  • Pantalla OLED

8. Fuentes de Energía y Regulación

Proveen energía al circuito y regulan el voltaje necesario para su funcionamiento.

  • Batería
  • Portapilas
  • Transformador
  • Regulador lineal

9. Módulos de Comunicación y Potencia

Facilitan la comunicación inalámbrica o el control de dispositivos de alto consumo.

  • Módulo Bluetooth
  • Módulo WiFi
  • Módulo GSM
  • Módulo GPS
  • Módulo RFID
  • Módulo relé
  • Módulo de potencia MOSFET
  • Módulo convertidor buck/boost

10. Placas, Conexiones y Terminales

Permiten montar, conectar y alimentar los componentes dentro de un circuito.

  • Placa protoboard
  • Placa PCB
  • Jack de alimentación
  • Conector jumper
  • Terminal de tornillo
  • Pin header
  • Socket DIP
